classON by gootyfer

in-Class Live Analytics for aSSessment and OrchestratioN

The classON project aims at improving the experience of teacher and students at lab sessions.

Dependencies

node packages: express, mongo, socket.io. client javascript libraries: jquery, jqueryUI.

Install

Install nodejs from nodejs.com Install mongodb Populate the user collection with your students And turn your class ON!
